# Enhancing Legal Information Access with Cutting Edge AI and RAG Technology

An AI-driven system designed to modernize legal research using Retrieval-Augmented Generation (RAG). This project helps users navigate complex legal documents with context-aware insights.

## 🚀 Features
* **RAG Integration:** Uses LangChain and Vector Databases for accurate information retrieval.
* **Smart Summarization:** Automated legal document analysis.
* **User-Friendly Interface:** Built with Python for ease of access.

## 🛠️ Tech Stack
* **Language:** Python,Java
* **AI Frameworks:** TensorFlow,PyTorch
* **Database:** PostgreSQL,MongoDB
* **Large File Support:** Git LFS (for model binaries and ffmpeg)

## ​💻 System Requirements
​* **Processor:** Minimum Intel Core i7 or equivalent.  
​* **RAM:** At least 16GB (32GB recommended for large datasets).  
​* **Storage:** Minimum 1TB SSD for fast data retrieval.  
​* **OS:** Windows 10/11, macOS, or Linux (Ubuntu recommended).

## 📋 How to Run
1. Clone the repository: `git clone https://github.com/Himani-komera/Legal-Info-RAG-System.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python manage.py runserver`
Once the server is running, access the platform at http://127.0.0.1:8000/ 

## ​📜 System Architecture
​The system follows a standard RAG pipeline:  
​* **PDF Ingestion:** Legal documents are processed and converted into embeddings.  
​* **Vector Database:** Embeddings are stored for semantic search and retrieval.  
​* **Pretrained LLM:** Queries are processed through a Large Language Model to generate enriched, context-aware outputs.
